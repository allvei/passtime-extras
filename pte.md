## Admin commands
```
sm_setteam         OR sm_st  // Set a client's team
sm_setclass        OR sm_sc  // Set a client's class
sm_ready           OR sm_rdy // Set a team's ready status
sm_debug_roundtime OR sm_drt // Debug: print team_round_timer info
```
## Backup admin commands
```
sm_enable_resupply <0|1> // Enable resupply functionality
sm_enable_respawn  <0|1> // Enable instant respawn
sm_enable_immunity <0|1> // Enable immunity and infinite ammo
sm_enable_saveload <0|1> // Enable save/load spawn functionality
sm_plugin_status   <0|1> // Show current plugin feature status
```
## Client commands
```
sm_save      OR sm_sv           // Save a spawn point
sm_load      OR sm_ld           // Teleport to saved spawn
sm_immune    OR sm_i            // Toggle invulnerability
sm_ammo      OR sm_a            // Toggle infinite ammo
sm_fov                          // Set your field of view.
+sm_resupply OR +sm_pt_resupply // Resupply inside spawn
```
